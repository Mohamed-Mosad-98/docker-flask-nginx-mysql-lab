# Task 2 - Docker Compose

## Deploy Stack

```bash
docker-compose up -d
```

## List Running Containers

```bash
docker ps
```

## View Running Services

```bash
docker-compose ps
```

## Stop Services

```bash
docker-compose down
```

## Restart Services

```bash
docker-compose restart
```

## Recreate Services

```bash
docker-compose up --build
```
